# Design 6: Aurora Waves (Vibrant Animated Gradient Mode)

A dynamic, immersive design containing colorful, shifting gradient meshes.

## Design Approach
- **Theme Concept:** Highlights energy, modernity, and a vibrant, alive community. Uses modern web animations to create an interface that feels responsive and organic.
- **Color Palette:**
  - Background Gradient: Deep Slate Blue (`#0f172a`)
  - Mesh Color 1: Warm Coral (`#fda4af` / `rgba(253, 164, 175, 0.15)`)
  - Mesh Color 2: Violet Glow (`#ddd6fe` / `rgba(221, 214, 254, 0.15)`)
  - Mesh Color 3: Sky Blue (`#bae6fd` / `rgba(186, 230, 253, 0.15)`)
  - Accent Color: Soft Rose (`#fb7185`)
- **Typography:**
  - Header & Body: *Plus Jakarta Sans* (Google Fonts) - A clean, friendly modern sans-serif with a slightly wide stance.
- **Key Visual Techniques:**
  - CSS `@keyframes` animations rotating background gradients slowly.
  - Transparent cards with thin white borders and subtle backdrop-filters.
  - Glowing, high-contrast primary CTA buttons with hover expansions.
  - Soft typography on contrasting dark colors.
- **Target Tenants:** Dynamic student associations, arts and music neighborhood districts, youth coalitions.
